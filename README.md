### Install

1. `git clone ..`
2. `composer install`
3. `cp .env.example .env`
4. edit .env (database)
5. `php artisan key:generate`
6. `php artisan migrate`
7. `php artisan db:seed`
8. `php artisan storage:link`
9. `npm install`

### Run
1. `php artisan serve`
2. `npm run dev`
